Vernier Caliper Measurement Experiment

Learn to measure length precisely using a virtual Vernier caliper with real scale reading.

Theory

Learn to measure length precisely using a virtual Vernier caliper with real scale reading. To measure the dimensions of a given object accurately using a Vernier caliper and to understand least count, zero error, and scale reading.

Formula

Principle: To measure the dimensions of a given object accurately using a Vernier caliper and to understand least count, zero error, and scale reading.

Aim

To measure the dimensions of a given object accurately using a Vernier caliper and to understand least count, zero error, and scale reading.

Apparatus

  • Vernier caliper
  • Cylinder
  • Coin
  • Rectangular block

Procedure

  1. Close the jaws completely and check for zero error.
  2. Place the object between the jaws.
  3. Slide the movable jaw until it gently touches the object.
  4. Read the Main Scale Reading (MSR) — the mark just before the Vernier zero.
  5. Find the Vernier Scale Division (VSD) that coincides with any main scale mark.
  6. Calculate: Reading = MSR + (VSD × Least Count).
  7. Record the observation and repeat for other objects.
